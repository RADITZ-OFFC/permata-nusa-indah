## Tips

- Format: MP4 (H.264) — paling kompatibel di semua browser
- Resolusi disarankan: 1280x720 (720p) atau 1920x1080 (1080p)
- Ukuran ideal per file: di bawah 30MB agar loading cepat
- Setelah taruh file di sini, langsung tampil di web tanpa perlu ubah kode apapun
