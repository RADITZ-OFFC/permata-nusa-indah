# Folder Foto Properti – Permata Nusa Indah Jayamulya

Taruh semua foto rumah/properti di folder ini.

## Cara Mengganti Foto

1. Copy foto Anda ke folder ini (format: JPG atau PNG, ukuran disarankan min. 800x600px)
2. Buka file `src/data/propertyImages.ts`
3. Ganti nilai URL atau path pada properti yang sesuai
4. Simpan — foto langsung terupdate di semua halaman

## Konvensi Penamaan File

Gunakan format: `<id-properti>_<nomor-foto>.<ext>`

Contoh:
- `1_1.jpg` → Properti ID 1, foto ke-1 (foto utama / thumbnail)
- `1_2.jpg` → Properti ID 1, foto ke-2
- `2_1.jpg` → Properti ID 2, foto utama
- dst.
